# Synthetic data included in the repository

All arrays, tables, maps, observations, hidden targets, and benchmark results
stored under `data/` and `results/` are generated by the source code in this
repository. No external, experimental, clinical, or third-party datasets are
included.

## Directory structure

```text
data/
├── synthetic/
│   ├── polarimetry_maps.npz
│   └── validation_challenge_01/
│       ├── public/
│       │   ├── challenge_manifest.json
│       │   ├── training_observations.csv
│       │   └── test_inputs.csv
│       └── reference/
│           └── hidden_test_targets.csv
└── example_orientation_maps/
```

The `public/` directory contains the information available to a calibration
algorithm. The `reference/` directory contains synthetic hidden targets used
only for scoring and regression testing.

## Reproducibility

The benchmark generators use explicit random seeds. Generated files can be
recreated by executing the corresponding notebooks or examples. Versioned
SHA-256 hashes are recorded in `release_manifest.json`.

## Scientific interpretation

These files support software verification, method development, and synthetic
predictive validation. They are not measurements of biological tissue and must
not be presented as experimental or clinical evidence.
