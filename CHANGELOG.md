# Changelog

All notable changes to this project are documented here.

## [1.0.0] — 2026-07-14

### Added

- stable package version and public API overview;
- comprehensive academic README with equations and figures;
- final reproducibility-audit notebook;
- GitHub Actions test, lint, compile, and build workflow;
- versioned release manifest with SHA-256 hashes;
- synthetic-data policy and reproducibility documentation;
- contribution, security, and issue-reporting guidance.

### Changed

- consolidated all distributed inputs and results under a synthetic-only data
  policy;
- promoted the project status from pre-release to stable version 1.0.0.

### Verification status

The release passes the automated unit-test suite, static linting, source
compilation, package build, example smoke tests, and the final release audit.

## [0.9.0] — 2026-07-14

- introduced a blind synthetic validation challenge with held-out loading
  protocols, repeated noise realizations, and architecture perturbation tests.

## [0.8.0] — 2026-06-10

- added weighted parameter identification, sensitivity, identifiability, and
  uncertainty analysis.

## [0.7.0] — 2026-05-28

- added multiple fiber families, orientation distributions, and graph
  regularization.

## [0.6.0] — 2026-05-11

- added coupled finite-element equilibrium and remodeling.

## [0.5.0] — 2026-04-24

- added total-Lagrangian Q4 finite-element equilibrium.

## [0.4.0] — 2026-04-04

- coupled synthetic polarimetry-like structural reconstruction to remodeling.

## [0.3.0] — 2026-03-30

- added synthetic azimuth and retardance-like structure reconstruction.

## [0.2.0] — 2026-02-13

- added spatially heterogeneous structural fields.

## [0.1.0] — 2026-01-10

- initial homogeneous finite-strain remodeling framework.
